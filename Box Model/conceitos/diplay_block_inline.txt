# display: block vs display:inline:

- Como as caixas se comportam em relação as outras

- Comportamento externo das caixas

block:
======
  - Ocupa toda linha colocando(empurra) o próximo elemento abaixo desse
  - width e height são respeitados
  - padding, margin, border funcionam normalmente



inline:
=======

- elemento lado a lado
- width e height não funcionam
- padding, margin, border funcionam apenas de forma horinzontal
